# Приложение для поиска и сортировки вакансий
### Приложение дает возможность сортировать по названию вакансий и словам в описании.
### Статус: _финальная стадия_
# Использование
### **КЛАСС HhAPI:**
### `load_vacancies` – метод загружающий список вакансий с указанным названием.
### `sorted_by_salary` – метод сортирующий список по убыванию зарплаты.
### `sorted_by_world` – метод сортирующий список ао словам в описании.
### **КЛАСС Vacancy**
### У класса есть магические `__repr__` и `__str__`, а так же методы для сравнения зарплаты.
### `add_new_vac` – метод для преобразования вакансий из Hh.ru в объект класса Vacancy
### КЛАСС LoadVacancy:
### `load_vacancy` – метод для загрузки вакансий в файл JSON.
### `get_vac` – метод, который получает вакансию по заданному критерию из файла.
### `clean_all` – метод, который очищает файл JSON.
# Тестирование
### Все функции проходят тестирование, при помощи `assert` и инструмента pytest.
### Для тестирования применяются инструменты `mock` и `patch`
